# Developer Setup

Canonical setup path for `gtcx-core` development.

## Prerequisites

- Node.js 20+
- pnpm 9+
- Rust toolchain (rustup)

## Install

```bash
pnpm install
```

## Rust Setup (Optional)

```bash
rustup update
cd rust
cargo build
```

## Common Commands

```bash
pnpm lint
pnpm format:check
pnpm typecheck
pnpm test
```
