# CI/CD

This document describes the CI/CD expectations for gtcx-core.

## CI Expectations

- Lint, typecheck, and tests must pass on every PR.
- Required checks mirror the quality gates in `docs/engineering/build-and-test.md`.
- Native bindings smoke test must pass for `@gtcx/crypto-native`.

## Release Expectations

- Quality gates are run and documented.
- UAT evidence logs are updated in `agile-pm/06 - planning/uat-evidence-log.md`.
- Release checklist is completed.
