# @gtcx/config

Shared configuration packages used across the workspace (eslint, typescript, tsup, tailwind).

## Scope

- `packages/config/eslint`
- `packages/config/typescript`
- `packages/config/tsup`
- `packages/config/tailwind`

These packages export preset configurations so packages can share consistent tooling.
