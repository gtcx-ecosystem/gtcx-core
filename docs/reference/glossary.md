# Glossary

- **DID**: Decentralized Identifier used for identity representation.
- **ZKP**: Zero-Knowledge Proof; proves a statement without revealing the witness.
- **UAT**: User Acceptance Testing; evidence-driven validation phase.
- **SLO**: Service Level Objective.
- **ADR**: Architecture Decision Record.
- **mTLS**: Mutual TLS authentication.
- **P2P**: Peer-to-peer networking.
- **Offline-first**: Design that assumes intermittent connectivity with deterministic sync.
