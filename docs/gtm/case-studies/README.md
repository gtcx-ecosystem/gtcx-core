> **Status:** Placeholder — awaiting first live pilot

This directory will contain verified case studies from live GTCX deployments. Each case study follows the template below and is validated against actual production data with participant consent.

## Case Study Template

| Section                     | Content                                                             |
| --------------------------- | ------------------------------------------------------------------- |
| **Executive summary**       | 2–3 sentences: who, what, where, outcome                            |
| **Participant profile**     | Entity name, sector, geography, regulatory context                  |
| **Problem statement**       | What verification gap existed before GTCX                           |
| **Solution deployed**       | Which GTCX packages, configuration, integration pattern             |
| **Implementation timeline** | Weeks from kickoff to production                                    |
| **Measurable outcomes**     | Quantified results (time saved, fraud reduced, compliance improved) |
| **Regulator interaction**   | Whether and how regulators were involved                            |
| **Lessons learned**         | What worked, what didn't, what changed in the product               |
| **Contact for follow-up**   | Named reference (with consent)                                      |
