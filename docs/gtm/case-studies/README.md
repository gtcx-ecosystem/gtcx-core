# gtcx-core Case Studies

> **Status:** Awaiting first pilot completion
> **Date:** 2026-05-27
> **Owner:** GTM Lead

This directory will contain verified case studies from live deployments of `gtcx-core` across African commodity markets.

## Template

Each case study follows this structure:

1. **Market context** — country, commodity, regulatory environment
2. **Problem before GTCX** — traceability gaps, compliance costs, fraud exposure
3. **Solution architecture** — which `gtcx-core` packages were consumed and how
4. **Implementation timeline** — weeks from kickoff to production
5. **Measurable outcomes** — cost reduction, time savings, compliance improvement
6. **Stakeholder quotes** — miner, agent, regulator, broker perspectives
7. **Technical evidence** — commit SHAs, test results, audit references
