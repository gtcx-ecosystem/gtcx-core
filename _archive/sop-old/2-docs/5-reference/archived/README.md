# Archived Reference

Historical documents, superseded designs, and deprecated references.

## What Belongs Here

- Specs or architecture docs that have been replaced by a newer version
- ADRs or decisions that were considered but rejected
- Legacy integration guides no longer applicable to the current system

## What Does Not Belong Here

- Active architecture docs → [`../../1-architecture/`](../../1-architecture/)
