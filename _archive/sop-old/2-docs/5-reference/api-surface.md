# API Surface Baseline

Points to the canonical API surface baselines and reports.

| Artifact      | Path                                | Description                                 |
| ------------- | ----------------------------------- | ------------------------------------------- |
| Baseline      | `quality/api-surface-baseline.json` | Committed baseline used for drift detection |
| Latest report | `quality/api-surface-report.json`   | Generated by `pnpm api:check`               |

## Usage

Validate current API surface against the committed baseline:

```bash
pnpm api:check
```

Update the baseline after an intentional API change:

```bash
pnpm api:update-baseline
```

## Policy

Per ADR-013, the API baseline gate runs in CI and blocks releases on unreviewed changes:

- Breaking diff → `major` version bump required
- Additive diff → `minor` or `major` version bump required
