# GTCX Protocols

Canonical standards and conventions for the GTCX ecosystem. These protocols define how we organize, build, and maintain our projects. Every GTCX repository follows these standards.

> **Protocols vs Templates:** Protocols define the RULES (this folder). Templates provide the FORMS to follow those rules (`/templates/`).
