# @gtcx/schemas

Runtime schema definitions for the Core12 framework (domains + controls). This package provides shared, typed data structures and helpers for compliance modules.

## Scope

- `CORE12_DOMAINS` data set
- `Domain`, `Control`, `FrameworkMapping` types
- Helper functions (`getDomain`, `getControl`, `getAllControls`)

- `packages/schemas/src/core12/*`
