# @gtcx/ai

AI‑integration stubs for traced operations and category logging. The full implementation lives in a separate repository; this package provides lightweight no‑op interfaces for compatibility.

## Scope

- `traced()` wrappers (no‑op)
- `withTrace()` helper (no‑op)
- `createCategoryLogger()` (no‑op logger)

## Notes

- This package is intentionally minimal and safe to use in any runtime.
