# Metrics

Templates for defining, tracking, and reporting on product and service metrics.

## Files

| File                                             | Purpose                                                                                     |
| ------------------------------------------------ | ------------------------------------------------------------------------------------------- |
| [performance-metrics.md](performance-metrics.md) | Service performance KPIs — latency, availability, throughput, error rates, DORA metrics     |
| [product-metrics.md](product-metrics.md)         | Product and business metrics — acquisition, engagement, retention, revenue, conversion, NPS |

## What belongs here

- Service-level indicators (SLIs) and service-level objectives (SLOs)
- Product health metrics (DAU, retention, conversion)
- Infrastructure and cost metrics
- Dashboard definitions and alerting thresholds
